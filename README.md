# grafikr-test-2.0

Repository containing a small project for my application at Grafikr.
The project mainly uses React, Tailwind and Vite (dev server and build tool).

### Installation

```
$ npm install
```

### Local Development

```
$ npm run dev
$ npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
```

### Build

```
$ npm run build
```
